Icons

The basic structure of the icon is an i with a class of ico and the name of the icon.

Some icons contain contenxual labels that are wrapped in a span. The span serves two purposes:

  1. Allows additional context to the icons
  2. Allows for 508 Adaptive compliance.

External Link

Using CSS rules in the base.less file. The rules will add an external link icon to all links that have a http://, https:// and .com in the URL except if eia.gov is in the file name. Additionally, special rules remove the external link from the .social (vertical slider) and .bx-viewport (sliders). To force an external link you can simply add a class="external-limk" to the anchor tag.
